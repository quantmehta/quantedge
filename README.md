## ğŸ’ Key Modules & Features

### 1. Autonomous Ingestion Engine (Phase 2 Focus)
*   **Dynamic Header Discovery**: Intelligent scanning of spreadsheets to find headers (Symbol, Qty, Price) regardless of row offsets or preamble noise.
*   **Flexible Schema Support**: Handles .csv, .xlsx, and .xls formats from various brokers (Groww, Zerodha, etc.).
*   **Bulk Persistence**: Optimized transaction-level processing for portfolios with 500+ instruments.

### 2. Multi-Factor Intelligence Hub
*   **Share-Level Analysis**: Recommendations are now individual (e.g., "TATA MOTORS: BUY") rather than just sector-wide.
*   **Contextual Reasoning**: Signals are generated by combining News Cycles, Policy Impacts (e.g., MSP Hikes, PLI Schemes), and Historical Performance.
*   **Decision Matrix**: Automatically identifies realized P&L and cost-averaging for sold/partially-sold holdings.

### 3. Event Intelligence (Portfolio Maximizer)
*   **Stress-Testing**: Define market events (Macro, Geopolitical, Sector) and simulate their impact.
*   **PnL-at-Risk**: Calculate exact downside exposure based on instrument sensitivity (Beta/Sector/Direct).
*   **Shock Matrix**: A real-time visualization of how different market shocks ripple through your portfolio.

### 4. Groww Integration & Live Context
*   **Live Data Sync**: Fetches current market prices and instrument details.
*   **Fund Manager Intel**: Python scripts analyze fund manager behaviors to feed into the suggestion engine.

## ğŸ› ï¸ Setup & Installation

### **Prerequisites**
*   Node.js v20.x
*   Python 3.10+ (with `pandas`, `numpy`, and `requests`)
*   SQLite3

### **1. Clone & Install**
```bash
git clone <repository-url>
cd decision-maker
npm install
pip install -r requirements.txt
```

### **2. Environment Configuration**
Create a `.env` file in the root:
```env
DATABASE_URL="file:./dev.db"
NEXT_PUBLIC_GROWW_MOCK="true" # Set to false for live integration
GROWW_API_KEY="your_api_key"
GROWW_API_SECRET="your_secret"
```

### **3. Database Initialization**
```bash
npx prisma generate
npx prisma db push
```

### **4. Start the Engine**
```bash
npm run dev
```
*Access the dashboard at `http://localhost:3000`*
