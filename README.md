## ğŸ› ï¸ Installation & Setup

### Prerequisites
*   **Node.js**: v20.x or higher
*   **Python**: v3.10+ (Ensure `python` and `pip` are in your PATH)
*   **Git**: Required for version control.

### 1. Clone the Repository
```bash
git clone <repository-url>
cd decision-maker
```

### 2. Install Dependencies
**Frontend/Backend (Node.js)**
```bash
npm install
```

**Analysis Engine (Python)**
```bash
pip install -r requirements.txt
```

### 3. Environment Configuration
Create a `.env` file in the root directory. This is critical for the system to function.

```env
# Database Connection
DATABASE_URL="file:./dev.db"

# Groww Integration (Live Data)
# Set to "true" to use mock data for testing without API keys
NEXT_PUBLIC_GROWW_MOCK="false" 
GROWW_API_KEY="your_api_key_here"
GROWW_API_SECRET="your_api_secret_here"

# System Configuration
NODE_ENV="development"
```

### 4. Initialize Database
Generate the Prisma client and push the schema to your local SQLite database:

```bash
npx prisma generate
npx prisma db push
```

### 5. Start the Engine
Launch the development server:
```bash
npm run dev
```
Access the dashboard at: `http://localhost:3000`

## ğŸ“– User Manual

### How to Import a Portfolio
1.  **Export**: Download your holdings report from your broker (Groww, Zerodha, etc.) as an Excel or CSV file.
2.  **Navigate**: Go to the **Import** tab in the QuantEdge dashboard.
3.  **Upload**: Drag and drop your file.
4.  **Verify**: The "Bulletproof" engine will auto-detect columns. Confirm the mapping (Symbol, Qty, Price) and click **Ingest**.

### How to Run a Decision Analysis
1.  **Open Chatbot**: Click on **Decision Maker** in the sidebar.
2.  **Start New Session**: Type "New Decision" or select a template.
3.  **Define Context**: "Should I sell my TATASTEEL shares?"
4.  **Set Alternatives**: "Sell Now", "Hold for 1 Year", "Buy More".
5.  **Define States**: "Market Crash", "Stable Growth", "Bull Run".
6.  **Compute**: The bot will ask for probabilities and payoffs, then output the optimal decision based on your chosen criteria (e.g., Maximin).
